# 🏠 House Price Prediction using Regression Models

This project predicts house prices using regression-based machine learning techniques on a real-world housing dataset. It demonstrates end-to-end data analysis, feature engineering, model building, and evaluation.

---

## 📌 Project Overview
The goal of this project is to analyze housing data and build regression models to understand which factors influence house prices and to predict prices accurately.

Models used:
- Linear Regression
- Polynomial Regression
- Ridge Regression (Regularization)

---

## 📊 Key Concepts Covered
- Exploratory Data Analysis (EDA)
- Feature selection and correlation analysis
- Handling missing values
- Data preprocessing and scaling
- Polynomial feature transformation
- Ridge regression and regularization
- Pipelines in scikit-learn
- Model evaluation using R² score

---

## 🧰 Technologies Used

- NumPy
- Pandas
- Matplotlib & Seaborn
- Scikit-learn
